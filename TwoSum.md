## Hinweis
* Man kann davon ausgehen, dass das Integer-Array aufsteigend sortiert ist

## Beispiel 1
```
Given nums = [2, 7, 11, 15], target = 9,

Because nums[0] + nums[1] = 2 + 7 = 9,
return [0, 1]

```
